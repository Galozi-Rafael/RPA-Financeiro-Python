# GnuCash-Automation-Python
Automatizar o aplicativo contábil Gnu Cash. Utilizando Python e a biblioteca Pywinauto

## Desenvolvimento

Essa automação será desenvolvida e evoluirá com o tempo.

Começarei usando os recursos básicos da biblioteca Pywinauto;
E usando o Inspect.exe para localizar os elementos na aplicação.

## Estágio

Por hora o bot somente inicia o GnuCash.
